Absolute positioning

One nuance with absolute positioning is that it will be locked relative to its closest positioned ancestor. If you forget to add a position rule to the parent item, (this is typically done using position: relative;), the browser will keep looking up the chain and ultimately default to the body tag.

Animation

The CSS selector contains properties of the animation itself:

  • animation-name
  • animation-duration
  • animation-fill-mode
  • animation-iteration-count
  • animation-timing-function

And the `@keyframes` property is information about what happens during the information e.g. change in position, colour, size, opacity which is set using syntax like:


@keyframes rainbow {
    0% {
    background-color: blue;
    top: 0px;
    }
    50% {
    background-color: green;
    top: 50px;
    }
    100% {
    background-color: yellow;
    top: 0px;
    }
}

Wrapping buttons in a fieldset tag

The fieldset tag surrounds the entire grouping of radio buttons to achieve this. It often uses a legend tag to provide a description for the grouping, which is read by screen readers for each choice in the fieldset element.

The fieldset wrapper and legend tag are not necessary when the choices are self-explanatory, like a gender selection. Using a label with the for attribute for each radio button is sufficient. 

            
<form>
    <fieldset>
        <legend>Choose one of these three items:
            <input id="one" type="radio"
                name="items" value="one">
            <label for="one">Choice One

            <input id="two" type="radio"
                name="items" value="two">
            <label for="two">Choice Two

            <input id="three" type="radio"
                name="items" value="three">
        <label for="three">Choice Three
    </fieldset>
</form>

Standardising datetimes

There is a time element which clarifies datetimes when dates are written colloquially or in some other form. It takes the form <time datetime="2020-01-05">5 January 2020</time>

Tab index

The tabindex attribute can be set to "0", positive or negative with different outcomes.

Setting it to "0" on a paragraph element ensures that the paragraph gets focused on when tabbing through the page.

CSS Flexbox

Adding display: flex to an element turns it into a flex container. This makes it possible to align any children of that element into rows or columns. You do this by adding the flex-direction property to the parent item and setting it to row or column. Creating a row will align the children horizontally, and creating a column will align the children vertically.

Other options for flex-direction are row-reverse and column-reverse.

Wrapping

flex-wrap takes one of nowrap, wrap, and wrap-reverse. Used with a size (width/height) it pushes children onto a new row/column.

The above flexbox properties all involve the parent container. You can also adjust child properties to manipulate the layout of the boxes.
